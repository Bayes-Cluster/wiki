Some Examples

In this page, we provide some script examples to illustrate how to submit a job. User can customize the script with these examples.

For MATLAB user

For now, MATLAB is no supported via the web portal, thus, user can only submit their MATLAB jobs via the terminal.Here is an example:

MATLAB job script
#!/bin/bash
#SBATCH -J test               # The job's name is test
#SBATCH -N 1                  # allocate 1 node
#SBATCH -p CPU-Compute-1      # submit the job to cpu partition
#SBATCH --cpus-per-task=4     # allocate 4 cpu cores
#SBATCH -t 1-00:00:00         # runtime: 1 day

source /Storage2030005000/home/user_name/environment/MATLAB-R2018a # activate the MATLAB-R2018a/MATLAB-R2020b PATH

# use MATLAB to run the "Test.m" file
$matlab -nodesktop -nosplash -nodisplay -r "Test"

matlab -nodesktop -nosplash -nodisplay means disable the MATLAB GUI and welcome page, -r means excecute the MATLAB command. The file name is “Test.m”, however, in the script, we need to remove the “.m”

Attention: CPU Node 1 (Compute2030005000) CANNOT run the MATLAB code with symbolic variables, please submit your task to other nodes

For GPU user

GPU job script
#!/bin/bash
#SBATCH -J gpu-job            # The job's name gpu-job
#SBATCH --cpus-per-task=8     # allocate 8 cpu cores
#SBATCH -p GPU-Compute-1      # submit the job to gpu partition
#SBATCH --gres=gpu:1          # allocate 1 graphic card
#SBATCH -t 1-00:00:00         # runtime: 1 day

source /Storage2030005000/home/user_name/environment/CUDA           # activate CUDA PATH
source /Storage2030005000/home/user_name/environment/anaconda-5.3.1 # activate anaconda PATH
source activate DeepLearning # activate the conda environment called DeepLearning

python gpu_task.py            # run the python script

For R user

It is easy for run the R script with SLURM, here is an example, assumes that we want to run job.R, with conda-based R environment:

R job script
#!/bin/bash
#SBATCH -J R-job              # The job's name R-job
#SBATCH --cpus-per-task=8     # allocate 8 cpu cores
#SBATCH -p CPU-Compute-1      # submit the job to CPU partition
#SBATCH --gres=gpu:1          # allocate 1 graphic card
#SBATCH -t 1-00:00:00         # runtime: 1 day

source /Storage2030005000/home/user_name/environment/anaconda-5.3.1
source activate environment_name

Rscript --vanilla job.R
